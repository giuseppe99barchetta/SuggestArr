Powerful Features

Everything you need to automate your media request workflow.

Multi-Media Server Support

Seamlessly integrates with Plex, Emby, and Jellyfin. Connect your existing library in minutes.

Intelligent Recommendations

Uses TMDb API to analyze watch history and find perfectly matched similar titles.

Automated Workflows

Automatically sends requests to Jellyseerr or Overseer without lifting a finger.

Real-Time Management

Monitor logs, manage cron schedules, and configure multi-user support from a sleek web UI.

User Selection

Choose specific users to initiate requests, allowing granular management and approval.

Config Validation

Automatically validates API keys and URLs during setup to ensure everything works perfectly.

Advanced AI

Next-Gen Intelligence

Supercharge your media server with cutting-edge Large Language Models.

LLM Provider Integration

Connect ChatGPT (OpenAI), Ollama, OpenRouter, LiteLLM, or any OpenAI-compatible endpoint with ease.

AI-Powered Search

Natural language search that understands what you want to watch. Personalized to your history and TMDb data.

AI Rationale Display

Don't just get suggested, get explained. Each recommendation includes an "AI Reasoning" section.

Enhanced Config

Granular settings for AI providers, including custom base URLs and improved visibility for all parameters.

Smart Context

Utilizes recent search history and automatic exclusion of already requested IDs for perfect results.

Arr Profile Sync

Import and distinguish download profiles directly from Sonarr and Radarr automatically.
